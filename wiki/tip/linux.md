# Linux Tip

## 자주쓰는 명령어들

    watch -n5 cat /proc/meminfo
    tail -f [filename]
    which [package name]
    du -h --apparent-size [directory name]
    df -h
    find . -name "*.xml" -exec grep -H "activity" {} \;
    find . -name "*.xml" | xargs grep "activity" --color=auto
    ipconfig getifaddr en0
